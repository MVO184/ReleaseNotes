Highlights

  • Aktualisierung der wesentlichen Abhängigkeiten (OpenSSL, cURL, SQLite) für bessere Sicherheit.
  • Verbesserte Windows-Integration und Stabilitätsverbesserungen beim Aufruf von Git via GUI-Clients.
  • Fehlerbehandlung für JSON-Eingaben verbessert (Server-Module geben nun immer valide JSON-Fehler zurück).

Änderungen (Auszug)

Core / Git

  • Git auf Version 2.55.x aktualisiert — Performance- und Sicherheitsfixes enthalten.
  • Verbessertes Handling für lange Pfade unter Windows (core.longPaths Korrekturen).

Tools & Abhängigkeiten

  • OpenSSL: Sicherheitsupdates und TLS-Verbesserungen.
  • cURL: Aktualisiert für bessere HTTPS-Kompatibilität.

Bugfixes

  • Behebt einen Fall, in dem fehlerhafte JSON-Payloads zu einem Server-Crash führten; jetzt wird HTTP 400 mit JSON-Fehlermeldung zurückgegeben.
  • Fix für Race-Condition beim parallelen Clone von Submodules unter Windows.

Bekannte Probleme

Kurz zusammengefasst:

  • Einige ältere GUI‑Clients zeigen noch falsche Pfad‑Trennzeichen an. Empfehlung: Git GUI aktualisieren.
  • Bei bestimmten AV-Setups kann der Installer länger zum Entpacken brauchen.
Hinweis: Solltest du auf einen Fehler stoßen, sammle Logs (git.exe --version, Systeminformationen) und öffne ein Issue im upstream-Repository.
